# Accommodation Booking Platform

A full-stack web application for browsing and booking accommodations.

##  Features
- User authentication
- Listings and booking system
- RESTful API backend
- MongoDB data handling

## Tech Stack
Node.js, Express, MongoDB, HTML, CSS, JavaScript

## Live Demo
https://accommodation-booking-platform-vcg8.onrender.com/listings

## ⚙️ Setup
git clone <repo>
npm install
npm start
